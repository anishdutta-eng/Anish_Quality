## 🔍 Quick Interpretation

### Good Connection ✅
```
RSSI: -60 dBm (strong signal)
SNR: 30 dB (clean signal)
Latency: 15 ms (fast)
MCS: 9 (high modulation)
Health: Excellent
```

### Problem Connection ❌
```
RSSI: -80 dBm (weak signal)
SNR: 10 dB (noisy)
Latency: 150 ms (slow)
MCS: 3 (low modulation)
Health: Bad
```

### What to Check First
1. **RSSI** - Am I close enough?
2. **SNR** - Is there interference?
3. **Latency** - Is network responsive?
4. **Recommendations** - What should I do?
