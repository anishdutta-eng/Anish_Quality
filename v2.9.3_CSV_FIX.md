## Changes Made

### 1. MCS Validation (NEW)
```python
# Validate and clean up MCS - must be 0-12
mcs_str = "N/A"
if mcs is not None:
    if isinstance(mcs, int):
        # Clamp MCS to valid range 0-12
        if 0 <= mcs <= 12:
            mcs_str = str(mcs)
        else:
            mcs_str = "N/A"  # Invalid MCS
```

**Output**: Only valid MCS values (0-12), invalid values show as "N/A"

### 2. Channel Utilization Cleanup (IMPROVED)
```python
# Clean up channel utilization - extract just the percentage number (no % or spaces)
cu_str = "N/A"
if cu is not None:
    if isinstance(cu, int):
        cu_str = str(cu)
    elif isinstance(cu, str):
        # Extract number from strings like "86%", "86 %", or "86"
        cu_clean = cu.strip().replace('%', '').replace(' ', '')
        match = re.search(r'\d+', cu_clean)
        if match:
            cu_str = match.group(0)
```

**Output**: `86`, `72`, `45` (numbers only, no spaces or % sign)

### 3. CSV-Safe Field Cleaning (IMPROVED)
```python
# Clean up SSID, Channel, BSSID - remove commas, newlines, and extra whitespace
ssid_clean = str(ssid).strip().replace(',', '_').replace('\n', ' ').replace('\r', ' ') if ssid else "N/A"
channel_clean = str(channel).strip().replace(',', '_').replace('\n', ' ').replace('\r', ' ') if channel else "N/A"
bssid_clean = str(bssid).strip().replace(',', '_').replace('\n', ' ').replace('\r', ' ') if bssid else "N/A"
```

**Output**: All newlines, carriage returns, and commas removed/replaced

### 4. PHY Mode Standardization
```python
# Clean up PHY mode - extract just the standard
phy_str = "N/A"
if phy and isinstance(phy, str):
    phy = phy.strip()
    if "802.11ax" in phy or "ax" in phy.lower():
        phy_str = "11ax"
    elif "802.11ac" in phy or "ac" in phy.lower():
        phy_str = "11ac"
    elif "802.11n" in phy or "ht" in phy.lower():
        phy_str = "11n"
```

**Output**: `11ax`, `11ac`, `11n`, `11a`, `11g`, `11b`

### 5. NSS Extraction
```python
# Clean up NSS - extract just the number
nss_str = "N/A"
if nss and isinstance(nss, str):
    nss = nss.strip()
    # Extract number from strings like "2" or "NSS: 2" or "2 streams"
    import re
    match = re.search(r'\d+', nss)
    if match:
        nss_str = match.group(0)
elif isinstance(nss, int):
    nss_str = str(nss)
```

**Output**: `1`, `2`, `3`, `4` (clean numbers)

### 6. Latency Formatting
```python
lat_str = f"{lat:.2f}" if lat is not None else "N/A"
```

### Field Specifications

| Field | Format | Example | Valid Range |
|-------|--------|---------|-------------|
| Iteration | Integer | 1, 2, 3 | 1+ |
| Timestamp_s | Float (2 decimals) | 0.37, 2.51 | 0+ |
| SSID | String (CSV-safe) | MyNetwork | Any |
| Channel | String (CSV-safe) | 44, 6g69/160 | Any |
| BSSID | MAC address | AA:BB:CC:DD:EE:FF | MAC format |
| RSSI_dBm | Integer | -45, -60, -75 | -100 to 0 |
| SNR_dB | Integer | 35, 25, 15 | 0 to 100 |
| Noise_dBm | Integer | -80, -90, -95 | -100 to 0 |
| TxRate_Mbps | Integer | 866, 1200, 286 | 0+ |
| Latency_ms | Float (2 decimals) | 12.50, 13.20 | 0+ |
| MCS_Index | Integer | 0-12 or N/A | 0 to 12 |
| PHY_Mode | String | 11ax, 11ac, 11n | Standard names |
| NSS | Integer | 1, 2, 3, 4 | 1 to 4 |
| ChannelUtil_% | Integer | 45, 67, 89 | 0 to 100 |
| Distance_m | Float (2 decimals) | 2.15, 5.43 | 0+ |
| Health_Status | String | Excellent, Good, Fair | Status |
